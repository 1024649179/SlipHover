SlipHover

Apply direction aware hover animation to images

  • ZIP File
  • TAR Ball

Example: title option

You can put html code in the attribute.

HTML:

<div id="container">
 <ul>
  <li>
   <a href="http://wayou.github.io/SlipHover/">
    <img src="../img/1.jpg" title="<p>Loyal Cao</p> <a href='http://www.google.com' style='color:gray;'>html is accepted</a>" alt="<p>image 1</p> <a href='http://www.google.com' style='color:gray;'>html is accepted</a>" >
   </a>
  </li>
  ...
  </ul>
</div>

JavaScript

$('#container').sliphover({
  title:'alt'//or 'title' or any other attribute
});
